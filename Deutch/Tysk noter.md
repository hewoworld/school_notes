## Tysk grammatik
 - På tysk begynder alle navneord med stort for at kende forskel på navneord og udsagnsord.
 - Z på tysk udtales som ts på dansk. Fx `Ich spreche zu schnell`, udtales som `Ich spreche tsu schnell`
-  Ei på tysk lyder som ai. For eksempel `Eis` på tysk udtales som `Ais`. `Ie` på tysk udtales som et forlænget i, fx. `Liebe` på tysk udtales rent faktisk som `Liibe`.

| Ental       | Endelse |
| ----------- | ------- |
| Ich         | -e      |
| Du          | -st     |
| Er/sie/es   | -t      |
| **Flertal** |         |
| Wir         | -en     |
| Ihr         | -t      |
| sie/Sie     | -en     |

## Nogle vigtige tyske oversætninger:
- `Ich komme aus` = `Jeg kommer fra`
- `Woher kommst du?`=`Hvor kommer du fra?`
